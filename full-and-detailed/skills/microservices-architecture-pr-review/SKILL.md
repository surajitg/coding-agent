# Microservices PR Review

## Purpose

Review PR changes for violations of microservice boundaries.

---

# Violations

Shared database usage.

Service accessing another service database.

Tightly coupled synchronous services.

---

# Output

Provide PR comments including:

Violation summary  
Affected services  
Recommended improvements
